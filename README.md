# Web AHP - Application d'aide à la décision multicritère

Application interactive utilisant la méthode **Analytic Hierarchy Process (AHP)** de Saaty.

## Fonctionnalités
- Choix du nombre de critères et alternatives (2 à 7)
- Évaluations numériques ou catégorielles pour chaque critère
- Matrice de comparaison par paires des critères
- Vérification de cohérence (RC < 0.1)
- Affichage de la meilleure alternative

## Utilisation en ligne
[Cliquez ici pour utiliser l'application](https://mmxavier-app.github.io/web-ahp)

## Utilisation locale
Ouvrir `index.html` dans un navigateur.
